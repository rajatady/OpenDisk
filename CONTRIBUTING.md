# Contributing

## Setup
1. Open `OpenDisk.xcodeproj` in Xcode.
2. Run tests before submitting changes.

## Standards
- Keep strict MVVM boundaries.
- Add tests for all behavior changes.
- Prefer protocol-based additions for service logic.
- Do not add external dependencies without discussion.

## Pull Requests
- Include summary, risks, and test evidence.
- Ensure CI is green.
